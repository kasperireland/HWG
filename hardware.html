All of our DirectAdmin servers run on Pure SSD NVME Drives to maximize speeds and minimize disk latency.

We use both the cPanel Software and the DirectAdmin Software to provide web hosting to our customers. cPanel has been the worlds industry leading web hosting control panel for decades, whilst DirectAdmin is just now getting more recognition and users. Some may say DirectAdmin is better than cPanel, but we think they are different and are good in their own ways. As of July 1st, 2021, we have officially moved to only providing DirectAdmin-based services.

Our Web Hosting Servers (DirectAdmin), feature an AMD Ryzen 9 3900X 12-Core Processor running at an average speed of 3792.876 MHz. 

Our servers, whether running DirectAdmin or cPanel, run CloudLinux OS, AlmaLinux OS, or Debian, depending on the server location, size, and purpose. All of our DirectAdmin servers are being migrated to AlmaLinux using custom code to limit resources per-user rather than using CloudLinux for that. We also encourage all of our DirectAdmin VPS customers which are using Cloudlinux to migrate to Almalinux.
